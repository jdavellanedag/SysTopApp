## Simple Study System Information App

This is a electron app for viw system usage

### Install

#### Install dependencies

```
npm install
```

or

```
yarn
```

### Usage

#### Run the app

```
npm run start
```

or

```
yarn start
```

#### Build the app (automatic)

```
npm run package-mac
npm run package-win
npm run package-linux
```

or

```
yarn package-mac
yarn package-win
yarn package-linux
```

#### Build the app (manual)

```
npm run build
```

or

```
yarn build
```
